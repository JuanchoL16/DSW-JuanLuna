Autoevaluación

Autoevaluación del sitio Web creado


En esta sección se busca realizar un Evaluación del sitio web creado teniendo en cuenta las mejoras que se pueden ejecutar en el mismo y las particularidades destacadas en el mismo, para lo cual se utilizarán diferentes criterios, entre ellos herramientas online como el de la página Validation Service que nos provee de los recursos necesarios para evaluar y determinar la calidad de nuestro sitio Web.


Propiedades básicas


  • Construir un sitio web explicando la temática
  • Desarrollar ejemplos que sirva para la explicación a realizar
  • Adjuntar contenido propio explicando la temática estudiada
  • Investigar sobre los diferentes temas para la presentación de los mismos
