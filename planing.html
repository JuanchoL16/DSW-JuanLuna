Materiales

  • Block: Se utiliza para que los elementos se muestren en bloques, a estos se les tiene que indicar todos los atributos para que se puedan mostrar como se desea, por ejemplo, se puede llevar los elementos por medio de parámetrós precisos por ejemplo el uso de la cantidad de pixeles usados para cada elemento.
  • Inline: Permite que los elementos se renderizen en línea con otros elementos, es decir, se adapata según el tamaño de la página y tomando citerios como la alineación para que se tome la posición correcta en cada vista
  • Inline-block: tiene un comportamiento combinado entre los dos anteriores y el texto fluye como si fueran elementos en línea, definiendo lo que se puede quedar estático y lo que no, también se tiene en cuenta los criterios estáticos o en línea.
